Who was awarded
THE NOBEL PRIZE IN LITERATURE?

Enter a year between 1901 and last year to find out who was awarded the Nobel Prize in Literature.

This app uses the Nobel Prize API to present facts about those who have been awarded the Nobel Prize in Literature. Learn more at Nobelprize.org. It also uses the LIBRIS API to get book titles by the laureates. Learn more at Libris.kb.se. Created by Loka Kanarp, Medieinstitutet 2018. GitHub repo.
